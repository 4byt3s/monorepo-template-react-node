# Monorepo template with reactjs and nodejs

## requirements

- [Lerna](https://lerna.js.org/)
- [Yarn](https://yarnpkg.com/)

## Commands

- Install dependencies: `lerna bootstrap`
- Run tests: `yarn test`
- Run development server: `yarn dev`
- Run production server: `yarn start`
- Build: `yarn build`


## Tools

- [Lerna](https://lerna.js.org/)
